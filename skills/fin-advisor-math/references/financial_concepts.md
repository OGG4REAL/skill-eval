# 投顾业务数学计算规范与术语

## 核心术语定义

### 1. 定投 (AIP - Automatic Investment Plan)
- **定义**: 投资者在固定的时间以固定的金额投资于指定的开放式基金。
- **计算频率**: 通常为每月一次，也可以是每周或每双周。
- **计算约定**: 
  - 默认采用**复利**计算。
  - 默认采用**期末投资**（即月末投入，当月不计息，次月开始产生收益）。

### 2. 复合年化收益率 (CAGR)
- **定义**: 一项投资在特定时间内收益的平均年增长率。
- **公式**: `(终值 / 现值)^(1 / 年数) - 1`
- **注意**: CAGR 并不反映中间过程的波动，仅反映起始和终点间的平均增速。

### 3. 最大回撤 (Maximum Drawdown)
- **定义**: 在选定周期内任一历史时点往后推，产品净值走到最低点时的收益率回撤幅度的最大值。
- **意义**: 衡量产品的抗风险能力，反映最极端情况下可能的亏损。

### 4. 夏普比率 (Sharpe Ratio)
- **定义**: 每承受一单位总风险，会产生多少的超额报酬。
- **计算**: `(投资收益率 - 无风险利率) / 收益率标准差`
- **意义**: 衡量风险调整后的收益，夏普比率越高，说明在承担相同风险的情况下，获得的收益越高。

## 计算约定与偏好

1. **收益率输入**: 除非特别说明，用户提到的“收益率”均默认为“年化收益率”。
2. **天数基准**: 计算时通常采用 365 天/年。
3. **货币格式**: 结果应保留 2 位小数，金额建议增加千分位分隔符。
4. **风险提示**: 在提供定投测算或收益测算结果时，必须包含以下提示：
   - "测算结果仅供参考，不代表未来实际收益。"
   - "定投不能规避基金投资所固有的风险，不能保证投资者获得收益。"

## 常用参数默认值
- **无风险利率**: 默认采用 2% (参考当前货币基金或国债水平)。
- **定投频率**: 默认"每月"。

## 对比分析输出规范

在进行投资方式对比（如"定投 vs 一次性投资"）时，必须遵循以下规范，避免产生误导性结论。

### 1. 输出顺序

对比结论必须按以下顺序呈现：

1. **期末总资产**（最直观的"最后我手里有多少钱"）
2. **实际盈利金额**（终值 - 投入总额）
3. **资金效率解释**（可选，解释为什么会有差异）

### 2. 避免常见错误

❌ **错误示例**：
> "一次性投资收益率更高（16.08% vs 18.00%）"

问题：数值表述与结论矛盾（16% < 18%），且混淆了"年化收益率"和"总收益"。

✅ **正确示例**：
> "一次性投资的**期末总资产更高**（113,227 元 vs 107,139 元），这是因为一次性投资让全部资金从第一天就开始参与市场增值。
>
> 虽然定投的**资金利用效率看起来更高**（因为平均投资时间更短），但由于总资金参与时间较少，最终积累的财富反而更少。"

### 3. 不同市场环境下的结论

| 市场环境 | 通常更优的方式 | 原因 |
|---------|---------------|------|
| 牛市（单边上涨） | 一次性投资 | 资金越早入场，越能享受全程增值 |
| 熊市（单边下跌） | 定投 | 分批买入降低平均成本 |
| 震荡市/先跌后涨 | 定投 | 低位买入更多份额（成本平均效应） |

### 4. 成本平均效应 (Dollar Cost Averaging)

定投的核心优势在于"成本平均效应"：
- **价格下跌时**：同样的金额能买到更多份额
- **价格上涨时**：之前低价买入的份额开始增值
- **结果**：平均买入成本可能低于市场平均价格

但这并不意味着定投在所有情况下都更好。在单边上涨的市场中，一次性投资让资金完整参与增值过程，最终财富更高。

### 5. 关键指标对比表格模板

在输出对比结论时，建议使用以下表格格式：

| 指标 | 一次性投资 | 定投 |
|------|-----------|------|
| 投资总额 | ¥100,000 | ¥100,000 |
| 期末总资产 | ¥113,227 | ¥107,139 |
| 实际盈利 | ¥13,227 | ¥7,139 |
| 收益率 | 13.23% | 7.14% |
| 更优方案 | ✅ | |

### 6. 结论陈述模板

>
> 这是因为【原因解释】。
>
> ⚠️ 测算结果仅供参考，不代表未来实际收益。
